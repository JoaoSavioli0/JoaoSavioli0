# João Savioli

**`Desenvolvedor FullStack`**

Meu nome é João Pedro Savioli, tenho 21 anos e sou de Itu, SP. Concluí o ensino médio na ETEC integrado à um curso técnico de Desenvolvimento de Sistemas. Hoje estudo Análise e Desenvolvimento de Sistemas na universidade Facens de Sorocaba. Meu foco de estudos se encontra em projetos fullstack, que incluem tanto a estruturação de APIs e bancos de dados quanto o design e desenvolvimento das telas exibidas ao usuário.
